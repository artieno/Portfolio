# Mitchelle Atieno — Portfolio

This project is a modern, responsive personal portfolio website built to present Mitchelle Atieno’s professional profile, skill set, featured work, and contact information. The site is lightweight, easy to customize, and designed to work well across desktop and mobile screens.

## Project Overview

The portfolio is a simple front-end web application that showcases:
- A short introduction and professional headline
- A skills section highlighting technical strengths
- A projects section for featured work
- Contact details for collaboration or networking

It is designed as a clean, visually appealing landing page that can be used as a digital resume or personal brand website.

## Tech Stack

### Frontend Technologies
- HTML5
  - Used for the page structure and semantic content sections such as header, main, footer, and navigation.
  - Provides the core layout for the portfolio.

- CSS3
  - Used for styling, layout, spacing, colors, and responsive design.
  - Implements a dark-themed visual style with modern UI effects such as rounded cards, spacing, and flexible grids.
  - Includes responsive media queries to adapt the layout for smaller screens.

- JavaScript
  - Adds small interactive enhancements such as:
    - A mobile navigation toggle button
    - Dynamic updating of the footer year
  - Keeps the site lightweight while improving usability.

### UI and Design Tools
- Google Fonts
  - The site uses the Inter font family for a polished and modern typography experience.

- External Images
  - The portfolio uses image assets from Unsplash to create a visually rich hero and project presentation.

### Development Environment
- Visual Studio Code
  - Primary editor for building and editing the project
- Git and GitHub
  - Used for version control and project sharing
- Python HTTP Server
  - Optional local development server for previewing the site in a browser

## Project Structure

The project is organized as follows:

- index.html — the main page content and structure
- css/styles.css — all design, layout, and responsive styling
- js/main.js — small interactive behavior for the site
- README.md — project documentation

## How to Run Locally

You can view the website locally in two simple ways:

### Option 1: Open directly in a browser
- Open index.html in your preferred browser.

### Option 2: Use a local web server
Run the following command from the project folder:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
