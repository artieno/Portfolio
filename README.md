# Mitchelle Atieno — Portfolio

This is a small, responsive portfolio site for Mitchelle Atieno. It contains a short bio, skills, one project, and contact links.

Open `index.html` in a browser to view locally, or run a simple static server:

```bash
# from this project folder
python -m http.server 8000
# then open http://localhost:8000
```

Files added:
- index.html
- css/styles.css
- js/main.js

You can customize the content (add projects, change copy, replace images) as needed.
